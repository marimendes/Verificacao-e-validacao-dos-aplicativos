
## Experiência do Usuário
#### Cenário 01

```

Funcionalidade: Botão voltar da tela de notificações 

Este cenário ocorre quando o usuário acessa a tela de notificações e tenta voltar pra a página anterior.
Entretanto o botão voltar é dar cor branca e se camufla muito bem na foto do usuário que é de cor preta e branca.
Tornando assim o botão quase invisivel.

DADO que eu acesse a tela de notificações, clicando no ícone se sino.
QUANDO eu decido voltar pra página anterior
ENTÃO eu não encontro o botão de voltar na tela

```

#### Cenário 02

```

Funcionalidade: Carteiras 

Este cenário ocorre quando o usuário acessa a tela chamada de "carteiras".
Entretanto a tela aparece vazia, sem informação.

DADO que eu faça o Login
E acessa a tela chamada "carteiras"
ENTÃO a tela de "carteira" aparece em branco, sem nenhuma informação.

```

Funcionalidade: Acessar aplicativo
